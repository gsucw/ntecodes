✦ About Neverness to Everness (NTE)

Neverness to Everness (abbreviated as NTE) is a free-to-play open-world anime RPG developed by Hotta Studio, the same team behind the popular open-world title Tower of Fantasy. Published by Perfect World, NTE launched globally on April 29, 2026 for PC, iOS, and Android with full cross-platform support — meaning your progress carries over seamlessly whether you play on a phone, tablet, or desktop.

Set in the sprawling, neon-lit city of Hethereau, NTE blends urban fantasy with the "urban anomaly" (都市怪谈) theme — a world where supernatural distortions called Anomalies seep into everyday reality. Players step into the role of an Appraiser (勘察者), a specialist who can detect and analyze these anomalies. You'll team up with a cast of Anomaly Hunters, each with unique combat abilities and backstories tied to the city's hidden mysteries.

The game features a real-time action combat system with elemental interactions, team-based combos, and fluid movement mechanics that let you parkour across rooftops and glide through the city streets. Beyond combat, Hethereau is packed with exploration activities — street racing, drone photography, arcade mini-games, and hidden collectibles that reveal the city's lore.

At launch, NTE features a substantial roster of gacha characters and weapons pulled through its banner system using Annulith, the premium currency. Daily commissions, Domain challenges, the Abyssal Tower endgame mode, and limited-time events give free-to-play players plenty of ways to earn pulls without spending money. The current launch banners feature limited-rate-up characters that rotate every few weeks.

If you're new to NTE, the most important thing you can do early is redeem all active gift codes — they give you a head start on Annulith, upgrade materials, and in-game currency that would otherwise take hours of grinding. → How to Redeem NTE Codes

Redeeming Neverness to Everness gift codes takes less than a minute. Follow these steps exactly — codes are case-sensitive and the redeem option only appears after you unlock the in-game mail feature.

01

Open the Game

Launch Neverness to Everness on your device and log in to your account. Make sure you're on the correct server (Global, Asia, Europe, or CN) — codes are server-specific and cannot be transferred between servers.

02

Unlock Mail

Progress through the opening tutorial until the in-game mail feature is unlocked. This typically takes 10–15 minutes of gameplay. The Redeem Code button is tied to the mail system, so you cannot skip this step. If the Redeem option doesn't appear, you haven't progressed far enough.

03

Open Menu

Tap the [Menu] button in the top-right corner of the main screen. On PC, you can press the Esc key to open the same menu. The menu layout is the same across all platforms.

04

Tap […] → Redeem

Tap the […] button (the "more" button), then select Redeem Code from the drop-down list of options. On some versions, this may also be labeled as "Gift Code" or "CDK Redemption."

05

Enter the Code

Type or paste the NTE code exactly as shown above. Codes are case-sensitive, so "NTE0429" and "nte0429" are different. Use the Copy button on each code card to copy the exact string to your clipboard and paste it directly — this eliminates the risk of typos or wrong capitalization.

06

Claim Reward

Tap Confirm. If the code is valid, you'll see a success message and the reward is delivered to your in-game mailbox instantly. Navigate to your mailbox to collect the items — note that mailbox items have their own expiration timer, so claim them within 30 days.

ℹ You must unlock the in-game mail feature by completing the tutorial before the Redeem Code option appears. Rewards are delivered to your in-game mailbox — claim them before mail expires (typically 30 days). If a code shows "Invalid" or "Expired," double-check for typos, verify the code is still on our active list, and confirm you're on the correct server.

⚠ Why NTE Codes Sometimes Fail

If a code isn't working, it's almost always one of these reasons:

Code Has Expired

NTE codes have expiry dates. Even codes posted recently can expire within hours. Always check the expired list first.

🔤

Typo or Wrong Case

NTE gift codes are case-sensitive. A single wrong letter or extra space causes the error. Use the Copy button above to avoid this.

🔁

Already Redeemed

Each NTE code can only be redeemed once per account. If you've used it before, you can't claim it again on the same account.

🌍

Region Restriction

Some Neverness to Everness codes are region-specific. A code released for the CN server may not work on global servers, and vice versa.

📶

Server Error

During maintenance or high-traffic periods, the redemption server can temporarily reject codes. Wait a few minutes and try again.

🔢

Redemption Limit Reached

Some limited-distribution NTE codes have a global cap on how many times they can be redeemed. Once that cap is hit, the code stops working for everyone.
